<h2><a href="https://www.geeksforgeeks.org/problems/repeating-character-first-appearance-leftmost/1?page=4&category=Strings&status=unsolved&sortBy=submissions">Repeating Character</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">Given a string <strong>s</strong>, return the index of the first character that appears more than once in the string. If there is no repeated character, return -1.</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><strong><span style="font-size: 18px;">Input: </span></strong><span style="font-size: 18px;">s = "geeksforgeeks"
<strong>Output: </strong>0<strong>
Explanation: </strong>We see that both 'e' and 'g' repeat as we move from left to right.But the leftmost is 'g' so we return leftmost index of 'g' that is 0.</span>
</pre>
<pre><strong><span style="font-size: 18px;">Input: </span></strong><span style="font-size: 18px;">s = "abcd"
<strong>Output: </strong>-1<strong>
Explanation: </strong>No character repeats so we return -1.</span></pre>
